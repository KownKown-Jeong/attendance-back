UserService:
    회원 가입 (User Registration):
        사용자로부터 회원 가입에 필요한 정보를 받습니다. (완료)
        - User 기본정보 및 가족정보 
        중복된 사용자(전화번호) 등을 확인합니다. (완료)
        비밀번호를 안전하게 해시 처리합니다. (완료)
        사용자 정보를 데이터베이스에 저장합니다. (완료)
    사용자 정보 조회 및 업데이트
        사용자 정보 조회 및 업데이트
    비밀번호 재설정 (Password Reset):
        사용자가 비밀번호를 잊어버린 경우 비밀번호 재설정 기능을 제공합니다.
        이메일이나 휴대폰 번호로 인증 코드를 발송하고, 인증 후 새로운 비밀번호를 설정할 수 있도록 합니다.
    사용자 역할 및 권한 관리: 사용자 정보 관리 (User Information Management):
        사용자의 프로필 정보를 조회하고 업데이트하는 기능을 제공합니다.
        사용자의 역할(role)이나 권한(permission)을 관리합니다.
AuthService:
    로그인 (User Login): (완료)
        사용자로부터 로그인 정보(사용자 이름, 비밀번호 등)를 받습니다.
        데이터베이스에서 사용자 정보를 조회합니다.
        비밀번호를 검증합니다.
        로그인 성공 시 JWT(JSON Web Token)과 같은 인증 토큰을 생성하여 반환합니다.
    인증 토큰 생성 및 검증 (완료)
    사용자 인증 관련 공통 로직 (완료)
EmailService 또는 SmsService:
    이메일 또는 SMS 발송
    회원 가입 시 인증 코드 발송
    비밀번호 재설정 시 인증 코드 발송
TokenService: (오늘)
    인증 토큰 생성 및 검증 (Token Verification):
        클라이언트에서 전달받은 인증 토큰을 검증합니다.
        토큰의 유효성을 확인하고, 만료 여부를 체크합니다.
        토큰에서 사용자 정보를 추출하여 요청에 첨부합니다.
    토큰의 유효기간 관리
    토큰 재발급 로직
